# Telegram Web Desktop Application

Полноценное **Desktop приложение** для доступа к настоящему веб-интерфейсу Telegram через MTProto прокси.

## Особенности

✅ **Настоящий Telegram Web** - загружается реальный web.telegram.org  
✅ **Работает внутри приложения** - открывается во встроенном браузере, а не в Chrome/Edge  
✅ **25 MTProto прокси** - автоматическое переключение при ошибке подключения  
✅ **Зацикленные попытки** - после последнего прокси → первый  
✅ **Обход блокировок** - оптимизировано для работы в Москве и РФ  
✅ **Красивый интерфейс** - дизайн в стиле Telegram Dark  
✅ **Готово к сборке в .exe** - работает на любом Windows компьютере  

## Список прокси

Включено 25 MTProto прокси:
- Основной: `84.252.74.108:443` (секрет: `d544dfc97e2434c0e410dda5d9cd41a3`)
- 4 официальных прокси Telegram
- 10 прокси для обхода блокировок в РФ
- 10 дополнительных прокси из открытых источников

## Установка и запуск

### Вариант 1: Запуск из исходного кода

```bash
# Установить зависимости
pip install flask pysocks requests pywebview

# Запустить приложение
python app.py
```

Приложение откроется в собственном окне!

### Вариант 2: Создание .exe файла

```bash
# Установить зависимости для сборки
pip install flask pysocks requests pyinstaller pywebview

# Создать EXE файл
python build_exe.py
```

После сборки в папке `dist/` появится файл `TelegramWebDesktop.exe`, который можно запустить на любом компьютере с Windows без установки Python.

## Как это работает

1. Приложение запускает локальный Flask сервер на `http://127.0.0.1:5000`
2. Открывается окно приложения с использованием библиотеки `pywebview`
3. Все запросы к `web.telegram.org` проходят через MTProto прокси
4. При ошибке подключения автоматически переключается на следующий прокси
5. После последнего прокси цикл начинается сначала

## Использование

1. Запустите приложение (`python app.py` или `TelegramWebDesktop.exe`)
2. Дождитесь открытия окна приложения
3. Нажмите кнопку **"🚀 Запустить Telegram Web"**
4. Войдите в свой аккаунт Telegram
5. Пользуйтесь полноценным Telegram через прокси!

## Преимущества перед браузерной версией

- 🚫 Не требует открытия браузера
- 🔒 Все соединения идут через MTProto прокси
- ⚡ Автоматическое переключение при блокировке прокси
- 🎨 Интегрированный интерфейс в стиле Telegram
- 💾 Работает даже при блокировке Telegram в регионе

## Примечание

Приложение использует настоящий веб-интерфейс Telegram (web.telegram.org), поэтому все ваши данные обрабатываются напрямую серверами Telegram, а не хранятся локально.
